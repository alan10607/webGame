## 益智遊戲 Puzzle Game
### <p><a target="_blank" href="https://alan10607.github.io/webGame/wall.html">城牆保衛者 Wall Defender</a></p>
- ***這是一個單機版本的策略地圖戰略遊戲***
- 主堡: 遊戲開始時在地圖隨機一處生成主堡, 主堡遭破壞即結束遊戲
- 守衛: 緊接著主堡或守衛的方塊可升級為守衛, 主堡與守衛比起一般方塊有額外效果
- 方塊分為以下: 遊戲開始時在地圖隨機一處生成主堡, 主堡遭破壞即結束遊戲與守衛比起一般方塊有額外效果
    - 土地:
        - 淺粉紅: 表尚無建造設施之新生土地, 無功能, 可升級成任意設施, 設施生命達30時自動在周圍生成
    - 設施:
        - 紅: 周圍散射　守衛效果: 射擊頻率提升
        - 藍: 瞄準連射　守衛效果: 連射數提升
        - 黃: 生命、回復速度提高　守衛效果: 更高的生命、回復速度

### <p><a target="_blank" href="https://alan10607.github.io/webGame/life.html">生命遊戲 Life Game(Conway's Game of Life)</a></p>
- [康威生命遊戲](https://zh.wikipedia.org/zh-tw/%E5%BA%B7%E5%A8%81%E7%94%9F%E5%91%BD%E6%B8%B8%E6%88%8F) 是一個零玩家遊戲
- 在一個二維方格世界, 每個方格代表一個活著或死亡的細胞, 一個細胞在下一刻的生死取決於周圍相鄰八個方格中的細胞狀態
- 若周圍活著的細胞過多, 則這個細胞會因資源匱乏而死去, 但若周圍活著的細胞過少, 該細胞也會因太孤單而死去
